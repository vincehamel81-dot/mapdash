# MapDash

A map-based driving game inspired by gumballquiz with street-based movement, clouds, and multiplayer modes.

## Setup

```bash
npm install
npm run dev
```

## Notes

- Uses React + Vite + Leaflet
- Fixed map controls with zoom buttons and driving-style orientation
- Static Quebec City region bbox defined in `src/config.js`
- Starter UI includes modes, lobby, cloud spawn, and wind display
